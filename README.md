# Meenit's Playground

Meenit's Playground is a mobile-first web game suite of three social deduction games:

- Odd One In
- Undercover
- Mafia

It is built with React, Vite, and TypeScript and is designed to be deployed as a static site on Render.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The production build will be output to the `dist` directory.
